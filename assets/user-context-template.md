# 圆酱的营养师膳食评价助手skill｜客户画像与个性化约束模板（营养师版，可选）

> 由营养师/营养团队在三日饮食记录分析前或分析中补充。没有的信息写“未提供”，不要脑补；未经客户授权的信息不要写入长期档案。

| 维度 | 内容 | 对分析的影响 |
|---|---|---|
| 年龄 / 性别 |  | 匹配 DRIs 年龄性别组；缺失时不做参考值差距 |
| 身高 / 体重 / 体重变化目标 |  | 能量、蛋白质、体重管理语境 |
| 孕哺 / 儿童 / 老年 / 慢病 / 用药 |  | 红线判断；必要时转专业人员 |
| 主要目标 | 减脂 / 增肌 / 控糖 / 改善早餐 / 减少外卖 / 改善便秘 / 其他 | 决定建议优先级 |
| 饮食限制 | 过敏原 / 素食 / 宗教 / 疾病限制 / 不耐受 | 决定替代食物和禁忌 |
| 喜好与厌恶 |  | 建议可执行性 |
| 烹调条件与预算 | 会不会做饭、是否常外卖、预算 | 决定“自己做/外卖优化/便利店方案” |
| 运动与活动水平 | 运动日/休息日、频率、强度 | 影响能量和蛋白分配解释 |
| 睡眠/加班/应酬/经期 |  | 解释夜宵、甜饮、情绪性进食等场景 |
| 补充剂/保健品 |  | 评估 UL 风险；不提供治疗剂量 |

## 最小补问集

如果时间有限，最多先问 5 个：

1. 年龄、性别、是否孕哺/特殊状态？
2. 身高体重和当前目标是什么？
3. 有无慢病、用药、过敏或明确不能吃的食物？
4. 这三天是否包含运动日、应酬日、熬夜或经期？
5. 最想先改善哪件事：体重、早餐、外卖、甜饮、便秘、血糖/血脂，还是其他？
