# 圆酱的营养师膳食评价助手skill｜可视化 Brief（可选）

> 仅当营养师需要“发给客户/内部复盘/做汇报/生成图表/做海报”时使用。默认不生成图片；默认先服务营养师判断，再生成客户可读版本。

## 1. 用途与受众

- 用途：营养师内部复盘 / 客户反馈 / 团队汇报 / 课程课件 / 社群科普二次改写 / 其他
- 受众：营养师本人 / 营养师团队 / 客户本人 / 家长 / 大众科普受众（需二次改写）
- 输出渠道：微信长图 / PPT / A4 PDF / 小红书图文 / 其他
- 语言与风格：温和专业 / 可爱亲切 / 临床简洁 / 科普风

## 2. 趋势图优先级

当圆酱/营养师要求“画趋势图”时，优先输出**可画数据表 + 图表建议**，必要时再交给图片/报告生成工具。趋势图不是必须画得花，而是要把三日变化看清楚。

优先级：

1. **一定可画（只依赖结构化记录）**：餐次完整度、优质蛋白出现餐次、蔬菜出现餐次、甜饮/酒精/奶茶次数、外卖/聚餐/重口味次数。
2. **条件可画（依赖份量估算）**：能量区间、蛋白/脂肪/碳水区间、膳食纤维粗估。必须画区间或误差条，不画精确点。
3. **严格条件可画（依赖 DRIs + 食物成分）**：DRIs 差距条形图/雷达图。只画已核验 DRIs 且摄入估算中高置信度的营养素。
4. **不建议画**：份量低置信度、DRIs pending、缺少年龄/性别/孕哺状态、只有“正常吃/吃了一些”的项目。

图表数据模板见：`assets/trend-chart-data-template.csv`。

## 3. 建议图表

| 图表 | 推荐图形 | 数据来源 | 是否可画 | 注意事项 |
|---|---|---|---|---|
| 三日餐次完整度 | 堆叠条形图/热力表 | 结构化表 | 是/否 | 未记录 ≠ 没吃 |
| 优质蛋白分布 | 餐次热力图/柱状图 | 餐次食物项 | 是/否 | 用出现次数/餐次分布优先于精确克数 |
| 蔬菜水果出现情况 | 分组柱状图 | 餐次食物项 | 是/否 | 蔬菜和水果分开呈现 |
| 外卖/甜饮/酒精频率 | 柱状图/标记线 | 原始记录 | 是/否 | 可作为行为入口 |
| 能量/宏量粗估区间 | 区间折线/误差条 | 估算表 | 条件允许 | 只能画区间，不画伪精确 |
| DRIs 差距条形图 | 横向条形图/雷达图 | 已核验 DRIs + 可靠摄入估算 | 条件允许 | 只画中高置信度项目 |

## 4. 趋势图输出格式

若直接输出给营养师或报告生成工具，按以下顺序：

1. **图表结论一句话**：这三天最明显的变化是什么。
2. **可画趋势图清单**：列出建议画哪几张图、为什么。
3. **图表数据表**：用 `assets/trend-chart-data-template.csv` 的字段输出。
4. **不可画内容说明**：哪些因为低置信度/缺 DRIs/缺份量不画。
5. **客户版图注**：温和解释，不羞辱客户。

## 6. 禁区

- 客户端不展示内部证据分级、prompt、未校对 DRIs 表；内部版可保留给营养师判断。
- 不把低置信度估算画成精确数值。
- 不用“缺乏症/疾病诊断”字样。
