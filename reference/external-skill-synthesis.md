## 已融合进 three-day-diet-record-analyzer 的增强层

### 1. 个性化画像层（来自 health-coach / 营养专家 Skill）

在分析前尽量收集但不强迫一次问完：

- 基本信息：年龄、性别、身高体重、孕哺/特殊状态。
- 目标：减脂、控糖、增肌、改善早餐、减少外卖、改善便秘、亲子饮食等。
- 饮食限制：过敏原、宗教/素食、疾病相关限制、药物/补充剂。
- 偏好/厌恶：喜欢/不喜欢的食物、烹调条件、预算、外食频率。
- 生活方式：活动水平、运动日/休息日、睡眠、加班/应酬、经期等。

输出时把建议写成“适合这个人的最小行动”，而不是通用口号。

### 2. 宏量目标与餐次分配层（来自 Nutrition-Macro-Calculator / health-coach）

当资料足够时可增加：

- 能量：估算 TDEE 或 DRIs EER 是否可用；若不可用，只保留趋势。
- 蛋白质：按 DRIs 参考 + 目标情境（运动/老年/减脂等）提示，但不越界成处方。
- 碳水：区分精制主食、全谷杂豆薯类、水果/甜饮来源。
- 脂肪：区分烹调油、油炸、坚果、动物脂肪、鱼类/不饱和脂肪来源。
- 餐次分配：优先检查早餐蛋白、午晚餐蔬菜、运动后恢复、夜宵/甜饮。

### 3. 趋势/相关性层（来自 nutrition-analyzer）

三日记录可观察但不能诊断：

- 餐次规律与能量波动。
- 运动日 vs 休息日饮食差异。
- 睡眠/加班/情绪与夜宵、甜饮、外卖的关系。
- 慢病/用药/孕哺/儿童老人等红线触发。
- 记录质量变化：是否能支持下一轮更精确计算。

### 4. 份量估计与中国饮食本土化层（来自 health-coach，结合本项目原则）

可作为追问/粗估辅助：

- 掌心肉类/豆制品 ≈ 一份优质蛋白线索。
- 拳头主食 ≈ 一份主食线索。
- 拇指油脂/坚果酱 ≈ 一份脂肪线索。
- 一碗饭、一份外卖、一杯奶茶、一盘清炒菜等必须注明“地区/餐馆/碗大小差异大”。
- 中国本土食品优先用：中国食物成分表、包装营养成分表、品牌官网/小程序数据、营养师团队内部核验表。

### 5. 可视化 brief 层（来自 nutrition-infographic-skill）

当用户需要发给客户或做汇报，可输出“可视化简报 brief”：

- 用途：客户反馈 / 内部复盘 / 朋友圈科普 / 课程课件。
- 受众：客户本人 / 家长 / 营养师团队 / 大众。
- 语言和风格：温和专业 / 可爱亲切 / 临床简洁。
- 核心图表：三日餐次完整度、蛋白分布、蔬菜出现次数、甜饮/外卖频率、DRIs 差距雷达/条形图（只放可信项目）。
- 禁区：不把低置信度数值画成精确柱状图；不展示内部 prompt 或未校对 DRIs 表。

## 执行原则

1. **方法可借鉴，数值要回源。** 外部 skill 的数字、阈值、食物数据默认不进入本项目正式内置表。
2. **三日记录先看模式，再看计算。** 计算服务于发现可行动入口，不追求伪精确。
3. **客户版少而准，内部版全而明。** 客户只看 1–3 个重点和最小行动；内部保留假设、证据等级、补问清单。
4. **个性化先问限制。** 过敏、疾病、孕哺、儿童、用药、宗教/素食、预算和烹调条件会直接改变建议。
5. **可视化只画可信内容。** 低置信度或未核验值只做文字提示，不做精确图表。
