# 全网交叉验证报告 — 中国食物成分表 docsqq PDF 是哪个版本

> 目标：不靠搜索摘要做结论；尽量抓到页面/文件比对。

## 抽到的强证据

### 证据 1：大黄米(黍) 349 kcal、蛋白质 13.6 g、脂肪 2.7 g 命中 1991 版

- 搜索 `"中国食物成分表" "1991" 谷类 大黄米 蛋白质 13.6 脂肪 2.7` 命中多条结果，其中 bmcx 营养网与 cnopendata 数据集均把"大黄米(黍)"的 蛋白质=13.6 g / 脂肪=2.7 g / 能量=349 kcal 归在 **1991 版/2002 版谱系**。
- 同一查询第二个结果 [大黄米[黍子]-食物营养成分查询平台 nlc.chinanutri.cn/fq/foodinfo/304.html](https://nlc.chinanutri.cn/fq/foodinfo/304.html) 是中国营养学会现行版本（推断为 2018 第6版标准版）的同名条目。对比：

| 字段 | 本 PDF（A 段 row 1） | chinanutri.cn `foodinfo/305` 黄米 | chinanutri.cn `foodinfo/304` 大黄米[黍子] |
|------|-------------------|-----------------------------------|------------------------------------------|
| 能量 (kcal) | **349** | 356 | （未确认） |
| 蛋白质 (g) | **13.6** | 9.7 | （未确认） |
| 脂肪 (g) | **2.7** | 1.5 | （未确认） |
| 水分 (g) | 11.3 | 11.1 | – |
| 维生素 B1 (mg) | 0.30 | 0.09 | – |

数值差异显著，几乎可以断定本 PDF 不是当前中国营养学会维护的官方在线数据；多个网络副本把 13.6 g 蛋白质 / 349 kcal 这组数归在 **1991 版**。

### 证据 2：营养素列范围与新版差距

通过 [2002 与 1991 版《食物成分表》的比较 — 国家食物与营养咨询委员会](https://sfncc.caas.cn/yyjk/37154.htm) 抓到：

- 1991 版「一般营养成分表」覆盖 **1358 个食物，28 个食物类别**
- 2002 版覆盖 **1506 个食物，21 个食物类别**；并首次新增 171 项叶酸、130 项碘、114 项大豆异黄酮、208 项血糖指数
- 编者：杨月欣、王光亚、潘兴昌；出版社：北京大学医学出版社（1991 版为人民卫生出版社）

本 PDF：1284 条食物条目，**两段都缺 磷/钾/镁/锌/硒/铜/锰/碘/氟** 等关键矿物质和微量元素。条目数 1284 < 1358（1991 完整版），说明本 PDF 是 **1991 版的子集或简表抄录**，而不是任何一版的完整数据。

### 证据 3：类别编码 11/21/22/31/41/51/61/71/86 = 1991 6-位编码体系

通过 [国内外食物编码发展现状及启示 — 中国公共卫生 2017](http://html.rhhz.net/ZGGGWS/html/20171031.htm) 抓到中国食物成分表沿用 **6 位双级食物编码**：前 2 位是大类，后 4 位是子类、产地、品种。常用大类映射为：

| 编码前 2 位 | 大类 | 在本 PDF 出现 |
|------------|------|--------------|
| 11 | 谷类 | ✓ |
| 12 | 谷类制品 | （需复核） |
| 21 | 干豆类 | ✓ |
| 22 | 豆制品 | ✓ |
| 31 | 蔬菜类 | ✓ |
| 41 | 菌藻类 | ✓ |
| 51 | 水果类 | ✓ |
| 61 | 畜肉 | ✓ |
| 62 | 内脏 | （需复核） |
| 71 | 蛋类 | ✓ |
| 72 | 乳类 | （需复核） |
| 84/86 | 油脂/调味、糖、酒 | ✓ |

本 PDF 的「类别」列就是这套 6 位编码里前 2 位的简化形式（但截断到 2 位无法区分子类，所以「11=谷类、21=干豆类、22=豆制品」这种**类内细分会丢失**）。

⚠️ 现行 WS/T 464—2015《食物成分数据表达规范》以及中国营养学会 2018 标准版第 6 版已切换到「**01 谷类 / 02 薯类 / 03 干豆 / 04 蔬菜 / 05 菌藻 / 06 水果 / 07 坚果 / 08 畜肉 / 09 禽肉 / 10 乳 / 11 蛋 / 12 鱼虾贝 …**」的两位连续编码。本 PDF 的 11/21/61 是**旧体系**，与新国标不一致。把这份 PDF 接进 skill 时，**类别码必须重新映射，不能直接合并到新数据。**

## 抽到的弱证据 / 未能抓到

- 国家公共卫生科学数据中心 `phsciencedata.cn/Share/ky_sjml.jsp?id=…` 列了一份「常用食物营养成分表」公开数据集，但页面只是目录，没有抓到字段说明；要拿数据要登录或申请。
- `cnopendata.com/data/nutrition.html` 自称"中国食物营养成分数据"，但页面只是商品介绍，无字段细节；商业付费数据集。
- 食典通 / 仪器信息网 / 知乎专栏 给出了 2018 第 6 版标准版的 **PDF 网盘下载链接**（[食典通-第1册](https://www.sdtdata.com/fx/fcv1/tsLibCard/195185.html)、[食典通-第2册](https://www.sdtdata.com/fx/fcv1/tsLibCard/195184.html)、[北医出版社官方页](https://www.pumpress.com.cn/en-shop/5514.html)、[知乎专栏](https://zhuanlan.zhihu.com/p/16279231053)）。**这是版权资料，不要在 skill 内直接打包；可以在文档里指引用户去合规渠道获取，或由圆酱用纸书数据手录入。**

## 结论 — 这个 PDF 大概率是什么

> **判断**：本 PDF 是 **1991 版《中国食物成分表》（中国预防医学科学院营养与食品卫生研究所 / 人民卫生出版社）** 的简表抄录，被某位使用者（创建者「长风·灼月」）在 2026-05-28 用 ET 表格 / WPS Office 导出 PDF 上传到腾讯文档。

支撑：
1. 大黄米/大麦/稻米等谷类条目能量 / 蛋白质 / 脂肪 数值与 1991 版完全吻合，与 2018 现行版不符。
2. 食物条目数（1284）落在 1991 版（1358）范围内、低于 2002 版（1506）。
3. 类别编码 11/21/31/61/71/86 属 1991 六位双级编码体系，与 2015 国标 WS/T 464 的新编码不一致。
4. PDF metadata 显示 creator=`ET 表格`，producer=`iOS Version 18.7.8 Quartz PDFContext`，创建时间=`2026-05-28 10:36:59 UTC` — 即一个 iPad/iPhone 上的 WPS 表格导出，本身不带版本信息字段。

⚠️ **不要在 skill 文档/输出里把这份 PDF 当成「最新中国食物成分表」对客户呈现**。必须标注：
- 数据来源：1991 版《中国食物成分表》抄录（非官方电子版，由腾讯文档分享者整理）
- 与现行 2018 第 6 版标准版的偏差：未含磷/钾/镁/锌/硒/铜/锰/碘等；蛋白质/能量数值未与新版核对
- 推荐升级：等圆酱拿到第 6 版标准版完整数据后替换

## 与其他公开来源做横向对比的可行路线

如果之后要做一次更严格的比对，建议下面 4 条线（按可信度排序）：

1. **中国营养学会官方查询平台** [`nlc.chinanutri.cn/fq/`](https://nlc.chinanutri.cn/fq/)。逐条抓取与本 PDF 同名食物的页面（如 `foodinfo/304`、`foodinfo/305`、…）做字段对比。注意该平台数据为现行版（推断 2018 第 6 版），数值会与本 PDF 不同。
2. **国家食物与营养咨询委员会** [`sfncc.caas.cn/yyjk/37012.htm`](https://sfncc.caas.cn/yyjk/37012.htm) "主要食物营养成分表"：可作为另一份 1991 谱系参考。
3. **公共卫生科学数据中心** [`phsciencedata.cn`](https://www.phsciencedata.cn/Share/ky_sjml.jsp?id=577e0301-ab65-432a-9bb7-a8342302e589) 申请「常用食物营养成分表」开放数据，确认其版本与许可。
